# app-messagerie
Application web de messagerie instantanée permettant une discussion de groupe en temps réel.

# Fonctionnalités
- Envoi et réception de messages en temps réel
- Discussion de groupe
- Interface web simple et interactive

# Technologies utilisées 
- HTML
- CSS
- PHP
- JavaScript
- Docker
- Docker-compose

# Installation
- Cloner le projet : git clone https://github.com/mak005/app-messagerie
- Lancer les conteneurs : docker-compose up
- Accéder à l'application via : http://localhost
